Verificação de Segurança para Manuseio de Baterias

Antes de manusear, verifique!

Responda as perguntas abaixo para saber se você está em condições seguras para operar com baterias. Ao final, o sistema indicará se há alguma recomendação a seguir.

1. Qual tipo de bateria você irá manusear?

O tipo define os riscos e cuidados específicos.

2. Onde o manuseio será realizado?

A ventilação é crucial, especialmente para baterias de chumbo-ácido.

3. Você está utilizando os EPIs corretos?

Luvas, óculos de proteção, avental e botina são obrigatórios.

4. Qual é o estado visual da bateria?

Inspecionar antes de manusear é passo obrigatório.

5. Há recursos de emergência disponíveis no local?

Extintor, lava-olhos e kit de primeiros socorros são itens obrigatórios.
